# MUG Z - Métele un Gol al Cáncer ⚽🎗️

¡Bienvenido al repositorio oficial de **MUG Z**, la campaña de prevención de cáncer de próstata más innovadora del norte grande! 🇨🇱

Este proyecto ha sido desarrollado por **Impacta Consultora** en colaboración con **Fundación SESP** para la Región de Tarapacá.

## 🌟 Características Principales

- **Cancha Interactiva**: Monitoreo en tiempo real de la participación de la comunidad.
- **MI FICHA (Perfil de Usuario)**: 
  - **Vitrina de Trofeos**: Gamificación para incentivar los chequeos preventivos.
  - **Banda Sonora MUG Z**: Una playlist exclusiva con temas alusivos a la campaña, permitiendo a los usuarios escuchar y valorar las canciones.
- **Primer Tiempo (Agendamiento)**: Sistema dinámico para buscar y agendar operativos de exámenes PSA en terreno.
- **Hit List MUG Z**: Un ranking social en la página de inicio que destaca los temas musicales más populares entre los hinchas.
- **VAR (Administración)**: Panel de control técnico para la gestión de usuarios y operativos.

## 🎵 Banda Sonora (Playlist)

La campaña cuenta con 10 temas musicales originales creados para motivar a la hinchada. Puedes encontrar los archivos de audio en `public/audio/`.
- Himno de la Prevención
- El Gol de tu Vida
- Ponte la Camiseta
- ¡Y muchos más!

## 🛠️ Stack Tecnológico

- **Frontend**: Next.js 14, TailwindCSS, Lucide Icons, Framer Motion.
- **Backend**: Server Actions de Next.js.
- **Base de Datos**: SQLite con Prisma ORM (ideal para despliegue rápido).
- **Diseño**: UI/UX premium con estética futbolera y efectos dinámicos.

## 🚀 Cómo empezar (Desarrollo)

1. Clonar el repositorio.
2. Instalar dependencias: `npm install`
3. Generar el cliente de Prisma: `npx prisma generate`
4. Ejecutar el seed para cargar datos iniciales (incluyendo la playlist): `npx prisma db seed`
5. Correr en modo desarrollo: `npm run dev`
